캐릭터

설정을 읽고 저장할 캐릭터 폴더를 선택합니다.

1. Expression 랜덤 이미지

표정 이름이 먼저 base 이미지 후보를 고릅니다. Set의 base 이미지는 이 후보 안에 있어야 Layer까지 이어집니다.

2. Set 기본 구성

Expression이 선택한 특정 base 이미지에 붙을 조합 단위입니다. Set 아래에 눈, 입, 소품 같은 Layer가 추가됩니다.

3. Set Base 이미지

이 Set의 기준 이미지입니다. 위에서 선택한 Surface expression의 랜덤 이미지 후보 중 하나와 같아야 합니다.

4. 저장

Expression 이미지 묶음과 Set 기본 구성을 각각 저장합니다. Layer는 이 Set이 정상 저장된 뒤 Layer 만들기에서 붙입니다.

Set과 expression을 조합할 준비가 됐어요.
