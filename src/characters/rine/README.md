# 리네 캐릭터

리네는 GhostNest의 풍부한 캐릭터 샘플이다.

## 포함된 예시

- profile
- dialogue lines
- expression image candidates
- DialogueScript surface
- mouth layer animation
- hitAreas

## 참고 포인트

`index.ts`의 `assets.surfaces["0"].layers.mouth`는 말할 때 입모양을 바꾸는 예시다.

히트박스 에디터에서 조정한 값은 최종적으로 `assets.hitAreas`에 반영해서 배포한다.
