Step 3 / Character Settings

캐릭터 상태 만들기

제작 홈

이 단계에서 하는 일

표정 이미지 후보 중 실제로 사용할 base 한 장을 골라 캐릭터의 기준 상태를 만듭니다. 파츠와 애니메이션은 이 상태 위에 붙습니다.

  1. 캐릭터를 선택합니다.
  2. 상태 ID를 정합니다. 처음이면 0으로 두어도 됩니다.
  3. 연결할 표정과 base 이미지를 고릅니다.
  4. 미리보기를 확인하고 저장합니다.

캐릭터

상태를 저장할 캐릭터를 선택합니다.

상태 기본 정보

상태는 표정, 기준 이미지, 이후 붙일 파츠를 묶는 단위입니다.

기준 이미지

선택한 표정에 등록된 base 후보 중 하나를 상태의 기준 이미지로 사용합니다.

저장

캐릭터 상태를 저장하면 다음 단계에서 눈, 입, 장식 같은 파츠를 이 상태 위에 추가할 수 있습니다.
