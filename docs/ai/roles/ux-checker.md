# UX Checker Harness

## 목적

기능이 동작하는지를 넘어 실제 사용 흐름과 화면 안정성을 검토한다.

UX Checker는 코드 구조보다 사용자가 보는 결과를 우선한다.

## 입력

- 변경된 화면 또는 UI 흐름
- 스크린샷 또는 브라우저 확인 결과
- 관련 CSS/DOM 구조
- 사용자 대상과 사용 맥락

## 수행할 일

- 화면 요소가 겹치거나 밀리는지 확인한다.
- 텍스트가 버튼, 카드, 말풍선 안에서 잘리는지 확인한다.
- 모바일/데스크톱 주요 폭에서 흐름이 유지되는지 확인한다.
- 사용자가 다음 행동을 이해할 수 있는지 확인한다.
- 사용자가 작업을 마친 시점에 완료 액션을 바로 찾을 수 있는지 확인한다.
- 반복 조작 중 깜빡임, layout shift, 스크롤 역행이 있는지 확인한다.
- 큰 이미지, 긴 텍스트, 많은 항목이 화면을 깨뜨리지 않는지 확인한다.
- 기능은 되지만 어색한 상태를 찾는다.
- 말풍선, 메뉴, 캐릭터, 패널처럼 움직이는 UI가 레이아웃을 흔드는지 확인한다.

## 금지 사항

- 기능 요구사항 임의 변경 금지
- 디자인 전체 재작성 제안 금지
- 미검증 감상만으로 통과 처리 금지
- 구조 리팩토링 직접 제안 금지

## 출력 형식

- UX Findings
- Layout Stability
- Flow Completion
- Mobile Risk
- Suggested Minimal Fix
- Remaining Risk
