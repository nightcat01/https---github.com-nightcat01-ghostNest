파츠 / 애니메이션 추가

제작 홈

이 단계에서 하는 일

캐릭터 상태 위에 눈깜빡임, 말할 때 입모양, 장식 같은 파츠를 얹습니다. 먼저 저장된 캐릭터 상태를 고른 뒤, 파츠 이미지와 위치를 정하세요.

  1. 캐릭터와 캐릭터 상태를 선택합니다.
  2. 파츠 종류와 레이어 ID를 고릅니다.
  3. 이미지 프레임과 위치를 미리보기에서 확인합니다.
  4. 저장해서 해당 상태에 파츠를 붙입니다.

파츠를 붙일 상태 선택

표정과 base 이미지가 연결된 캐릭터 상태를 고른 뒤, 그 상태에서만 동작할 파츠를 만듭니다.

저장 설정

이미지는 base 또는 parts 폴더에 저장하고, 선택한 캐릭터 상태의 layers 항목에 연결합니다.
