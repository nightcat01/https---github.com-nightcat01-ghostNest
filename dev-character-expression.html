제작 홈

이 단계에서 하는 일

표정 이름에 base 이미지 후보를 묶습니다. 여러 장을 선택하면 같은 표정 안에서 랜덤 후보로 쓸 수 있어요.

  1. 캐릭터를 선택합니다.
  2. 표정 이름을 고르거나 새로 만듭니다.
  3. 이 표정에 사용할 base 이미지 후보를 선택합니다.
  4. 미리보기를 확인하고 저장합니다.

캐릭터

표정 이미지를 저장할 캐릭터를 선택합니다.

표정과 이미지 후보

여기서 고른 이미지는 다음 단계에서 캐릭터 상태의 기준 이미지가 됩니다.

이미지 등록

새 이미지를 바로 캐릭터 에셋 폴더에 저장합니다. 한 장짜리 완성 이미지처럼 쓸 scene도 표정 후보로 연결할 수 있습니다.

저장

선택한 base 이미지 목록을 캐릭터의 표정 후보로 저장합니다.
