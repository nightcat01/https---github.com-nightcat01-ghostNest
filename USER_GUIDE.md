# GhostNest 사용자 가이드 (User Guide)

GhostNest는 웹 브라우저에서 바로 만나볼 수 있는 '웹 고스트(나니카/우카가카)' 런타임 환경입니다. 
귀엽고 친근한 캐릭터가 화면 한구석에 머물며 여러분과 대화를 나누고 소소한 상호작용을 제공합니다.

## 🌟 기본 조작법

### 캐릭터와 상호작용하기
- **클릭 (Click):** 캐릭터를 가볍게 클릭하면 반갑게 인사를 건네거나 짧은 대사를 합니다.
- **쓰다듬기 (Touch):** 머리, 얼굴, 몸통 등 특정 부위를 클릭하면 각 부위에 맞는 특별한 반응과 표정 변화를 볼 수 있습니다.
- **더블 클릭 (Double Click):** 캐릭터를 더블 클릭하면 **[관리 메뉴]** 가 열립니다.

### 관리 메뉴 활용하기
관리 메뉴에서는 고스트의 다양한 기능을 직접 실행할 수 있습니다.
1. **한마디:** 고스트에게 랜덤한 대사를 요청합니다.
2. **운세 실행:** 고스트가 오늘의 운세를 봐줍니다. (플러그인이 설치된 경우)
3. **말풍선 테마:** 대화창(말풍선)의 분위기(기본, 부드러운 테마, 다크 테마 등)를 바꿀 수 있습니다.
4. **점프:** 고스트가 가볍게 뛰는 애니메이션을 보여줍니다.
5. **숨기기:** 화면에서 고스트를 잠시 숨깁니다.

## 💬 대화 및 상태

- **방치 모드 (Idle):** 일정 시간(기본 약 18초) 동안 아무런 상호작용이 없으면 캐릭터가 가만히 대기하는 상태가 됩니다.
- **랜덤 대사 (Random Prompt):** 가만히 놔두면 가끔씩 고스트가 먼저 혼잣말을 하거나 말을 걸어옵니다.
- **UI 감시 (Hover):** 화면에 있는 특정 버튼이나 영역(이벤트 로그 등)에 마우스를 올리면 고스트가 해당 UI에 대해 참견하거나 설명을 해줍니다.

## ⚙️ 화면 UI 설명
- **말풍선:** 고스트가 하는 말이 출력됩니다.
- **상태창(디버그/테스트 모드일 경우):** 현재 고스트의 표정, 남은 방치 시간, 다음 랜덤 대사까지의 시간 등을 확인할 수 있습니다.
- **이벤트 로그:** 고스트와 어떤 상호작용을 했는지 기록이 남습니다.

이제 화면 속 고스트와 즐거운 시간을 보내보세요!
